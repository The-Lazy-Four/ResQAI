Crisis Portal
What this module does
  • Provides a comprehensive dashboard for immediate public emergency tracking and response.
Key features
  • Real-time emergency detection and alerts
  • SOS triggering with AI voice + guidance
  • Live map and evacuation support
How to use
  • Access the Command Center during an incident.
  • Follow the AI-generated real-time instructions.
  • Use the SOS trigger for immediate critical assistance.
Real-life benefit
  • Best for public and large-scale environments to coordinate rapid evacuations and mass alerts.
EcoPlus (Hotel Module)
What this module does
  • A specialized interface designed to handle emergencies within hospitality and high-rise structures.
Key features
  • Designed for hotels and resorts
  • AI CCTV monitoring (fire, smoke, weapon detection)
  • Staff coordination and guest safety
  • Multi-floor emergency management
How to use
  • Monitor the smart dashboard for automated CCTV alerts.
  • Deploy staff dynamically across different floors.
  • Broadcast safety messages directly to guest room devices.
Real-life benefit
  • Saves lives by allowing hotel security to pinpoint threats instantly and manage localized evacuations safely.
Custom Rescue Builder
What this module does
  • A flexible platform enabling organizations to build private, tailored emergency response systems.
Key features
  • Admin builds custom emergency systems
  • Users join via access code
  • Personalized safety guidance
  • Suitable for schools, offices, campuses
How to use
  • Admins create the system layout and invite users via unique codes.
  • Users input their access code to sync with the organization's network.
  • Follow the custom procedures designated by the admin during a crisis.
Real-life benefit
  • Empowers schools and corporate campuses to maintain closed-loop safety networks specific to their building layouts.
